Данный модуль разработан в рамках выпускной квалификационной работы на тему:  
**«Разработка системы для автоматизации процесса поиска и управления практикой студентов»**.

Модуль является составной частью автоматизированной системы организации практики и предназначен для наполнения базы данных актуальными предложениями о стажировках.

## Назначение
Модуль автоматически собирает IT-стажировки с платформы HeadHunter, сохраняет информацию о вакансиях и работодателях в PostgreSQL, а также поддерживает актуальность данных (деактивирует закрытые вакансии).

## Запуск
1. `pip install -r requirements.txt`
2. В `config.py` укажите свои данные
3. `python hh_parser.py`
